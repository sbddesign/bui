Model Context Protocol server for Bitcoin UI Kit components

MCP Endpoint

POST /mcp

Available Tools

instructions
Learn how to use the Bitcoin UI Kit and these tools
list_components
Get a list of all available components
get_component
Get detailed info about a specific component

Example Usage

// Initialize
POST /mcp
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "initialize",
  "params": {
    "protocolVersion": "2024-11-05",
    "clientInfo": { "name": "my-client" }
  }
}

// Call a tool
POST /mcp
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "tools/call",
  "params": {
    "name": "list_components",
    "arguments": {}
  }
}
